# LifePulse v4 — Complete Vision

## Inspiration
Life Reset app + Tinder/Hinge swipe UX + Pokemon card gamification + RPG stats

## Pages & Tabs

### 1. TODAY (Swipe Cards — like Tinder/Bumble)
- Full-screen swipeable cards, ONE habit per card
- Each card has an anime/illustration background image (user can add images)
- Day selector at top (past 5 days)
- Progress dots showing which card you're on
- Card shows: habit name, streak badge, input area
- **Card-specific inputs (scroll picker style like Life Reset):**
  - Pages: vertical scroll picker "10 pages"
  - Water: scroll picker "2.1 L" with bottle visuals
  - Sleep/Wake: scroll picker "10 AM" with Night Shift toggle
  - Workout duration: scroll picker "15 minutes"
  - Boolean: big confirm button
  - BP: dual number input
- Bottom: big orange "Confirm" button per card
- Overall streak visible
- Can create new card/habit from here

### 2. PROGRESS (Row List → Pokemon Card)
- **List view:** Simple rows — icon + habit name + target + status toggle
- **On click → Pokemon Card page** for that habit:
  - Anime/illustration card header
  - Charts showing progression (line chart, "Life Reset vs Without" comparison)
  - Git-like heatmap/contribution graph
  - Streaks (current, longest)
  - Strength/stats for that specific habit
  - "Progressively more difficult" week-by-week goals

### 3. SCORE (RPG Power Levels)
- Anime character background (right side)
- **Level badge** (Level 1, Level 35, etc.) with XP bar
- **5 personality stats** — RPG-style big numbers:
  - Wisdom (brain icon)
  - Confidence (person icon)
  - Strength (dumbbell icon)
  - Discipline (book icon)
  - Focus (target icon)
- Each stat has green arrow + score
- "See potential rating" button
- **Suggested habits** (swipeable cards like Tinder):
  - Card with anime art + habit name (MEDITATE, JOURNAL)
  - "93% fit your goal" badge
  - Stat boosts shown (Clarity +20%, Mood +15%)
  - Scientific study citation
  - Swipe to decline/add
- **Achievement cards**

### 4. ALARMS (Centralized)
- Single page to manage ALL habit reminders
- Per-habit alarm time + days configuration
- Easy toggle on/off per habit

### 5. PROFILE
- User info, settings, theme toggle, sign out

## Navigation
- Desktop: Icon rail sidebar (like current v4 tag, but refined)
- Mobile: Bottom nav bar
- 5 tabs: Today, Progress, Score, Alarms, Profile

## Design System
- Dark mode primary (true black)
- Green primary (#16A34A / #22C55E)
- Orange accent for CTAs (like Life Reset)
- Anime/illustration art style for cards
- Scroll picker inputs (vertical ruler-style)
- Gaming/RPG typography for stats
